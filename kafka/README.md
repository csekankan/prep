A hands-on, interview-focused tutorial series that takes you from Python OOP fundamentals through advanced design patterns and real-world concurrency — everything you need to ace Low-Level Design rounds and write production-grade Python.

Whether you're preparing for system design interviews at top tech companies or leveling up your day-to-day engineering craft, this series gives you the mental models, code patterns, and practice problems to get there.

## ✅ Prerequisites

Before starting this series, you should be comfortable with:

- **Python 3.10+** syntax (type hints, match statements, walrus operator)
- Basic data structures (lists, dicts, sets, deques)
- Functions, closures, and first-class functions
- Virtual environments and `pip`
- Running Python scripts and using a REPL

No prior design patterns or concurrency knowledge is assumed — we build from first principles.

## 🛠️ Running the Code

All code examples are self-contained and can be run with:

```bash
python3 <filename>.py
```

Some concurrency examples require Python 3.11+ for `asyncio.TaskGroup`. The series uses no external dependencies unless explicitly noted.
