# Final Advice

Do NOT solve these like:
- memorize solution
- copy pattern
- watch explanation only

Instead ask:

1. What information flows upward?
2. What state fully describes problem?
3. Can I transform problem?
4. Is this edge contribution?
5. Is this interval
